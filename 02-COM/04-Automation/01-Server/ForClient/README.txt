
NOTES FOR USAGE
-------------------------------------------------------------------------------------------------------------------

1. Copy AutomationServer.dll, AutomationProxyStub.dll and AutomationTypeLibrary.tlb to C:\\Windows\\System32

2. Open CMD in Administrator Mode. Give the following command to register the AutomationProxyStub.dll
	- regsvr32.exe AutomationProxyStub.dll

3. Register the AutomationServer.dll and AutomationTypeLibrary.tlb using the AutomationServer.reg
