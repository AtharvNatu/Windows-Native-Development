1. Window's left top should be aligned with the top-left of the desktop

2. Window size should be 800x600, set using macros

3. Window should be black in color

4. Custom Icon

5. Dialog box should be displayed when space key is pressed, and should not overlap with parent window

6. The dialog box should have 2 push buttons on the bottom right - OK and CANCEL

7. When OK button is pressed, "OK" Button Clicked message should be displayed in green color on parent window
	If CANCEL button is pressed, "CANCEL" Button Clicked message should be displayed.
